## 📂 Repository Structure

### 🧪 Hands_On_Guide (카카오 클라우드 사용 가이드)
| 디렉토리 | 설명 |
|----------|------|
| Api_cli | api token을 발급받기 위한 CLI 커맨드 정리|
| IAM | IAM token을 발급받기 위한 CLI 커맨드 정리 |
| Monitoring | Monitoring agent 설치를 위한 커맨드 정리 |
---

### ⚡ Tutorial (실습 & 실험)
| 디렉토리 | 설명 |
|----------|------|
| Big Data | Hadoop Eco, Data Catalog 등 각종 코드 정리 |
| Compute | Mongodb 및 WAS 등 각종 코드 정리 |
| Container | 각종 Container 관련 오픈 소스 코드 정리 (Rancher, S3 등) |
| Dev_Tools | 각종 Gitops를 활용한 튜토리얼 정리 |
| Machine Learning AI | AI 관련 코드 정리 |
| Network_content_delivery | CDN관련 오픈 소스 관련 튜토리얼|
| Observability | 관측 관련 오픈 소스 관련 튜토리얼 |
